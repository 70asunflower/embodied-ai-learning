# 台大机器人学之运动学——林沛群

> 台大机器人学专项课程，由林沛群教授讲授。
> 原版 Coursera: https://www.coursera.org/learn/robotics1
> B 站搬运: https://www.bilibili.com/video/BV1v4411H7ez

## Overview

台大机器人学专项课程，讲解机器人运动学与动力学的基础理论。B 站为国内搬运版本，配有课件和书籍资料。

## Course Content

- **运动学建模**: 位置、姿态、坐标系变换
- **正运动学**: 从关节角度到末端位姿的映射
- **逆运动学**: 从目标位姿反推关节角度
- **速度运动学**: Jacobian 矩阵与速度映射
- **动力学基础**: 力/力矩与运动的关系
